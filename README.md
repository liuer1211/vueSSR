# vue_ssr

## Build Setup

```bash
# 下载依赖
$ npm install

# 运行访问 localhost:3000
$ npm run dev

# 打包
$ npm run build
$ npm run start

# 静态文件
$ npm run generate
```

For detailed explanation on how things work, check out [Nuxt.js docs](https://nuxtjs.org).
